# SmartElectrons (IoT Dashboard)
An IoT control dashboard created as a personal project inspired by smart home automation systems.

## Features
- Monitor temperature and device status in real time  
- User-specific device control and authentication  
- Responsive web interface (Bootstrap + JavaScript)  
- Basic backend logic using REST APIs  

## Tech Stack
- Java (Spring Boot)  
- Angular, TypeScript  
- REST API, MySQL  
- HTML, CSS, Bootstrap  

## Description
This project was developed locally to simulate a smart device monitoring dashboard.
It allows users to view active devices, temperature, and turn them on/off through a simple interface.
